# Schedule-Moodle-Slack
Schedule + Moodle + Slack через термінал




--- 
Проєкт "S-M-S"
Schedule + Moodle + Slack через термінал
  

Функціонал

1. Людина стартує програму
2. Спочатку вискакує логін 
   - буде база данних студентів
   - можна реєструватись як простий користувач
   - початково один Грант Адмін який може надавати права іншим адмінам


2. Дальше менюшка на вибір того куди хочеш увійти (плюс розподіл на юзерів)

   - Moodle
   - Slack
   - Schedule


3. Schedule (максимальний пріоритет)
   - якщо студент то ти просто дивишся пари
   - якщо ти адмін то ти можеш виставляти і видаляти пари в разклад
   - при видаленні пари коли ти заходиш то тобі вона може підсвічуватись додатково що вона перемістилась
   - також як ідея можна зробити щось типу як вибір годин на офіски з дискретки (кожен вибирає час який йому зручний і потім це все сумується)
 
   
4. Moodle (середній пріоритет)
   - для студента буде можливість вибрати якісь предмет та заенролитись на нього
   - на предметі будуть якісь матеріали які може викладати адмін (викладач)
   - якщо викладач видалив щось - то це видалилось у всіх людей
   - можливо ще якісь ідеї


5. Slack (мінімальний пріоритет)
   - поки студент не заенролився на курс нема допуску
   - якщо студент заенролився на курс то можна зайти і буде чат з відповідним курсом де помна писати
