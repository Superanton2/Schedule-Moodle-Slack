**S-M-S** — це програма на Python для управління розкладом занять в університеті через термінал.
Система дозволяє студентам реєструватися, переглядати свій розклад у візуальному форматі, записуватися на курси та виходити з них. 
Також передбачено роль адміністратора для управління глобальним розкладом.

# Функціонал
**Для Студентів (User)**
* **Авторизація:** Безпечний вхід та реєстрація нових користувачів.
* **Інтерактивний розклад:** Перегляд тижневого розкладу у вигляді структурованої таблиці прямо в консолі.
* **Enroll (Запис):**
  * Інтелектуальна система запису на дисципліни.
  * Автоматична перевірка конфліктів у часі (накладання пар).
  * Перевірка доступності груп (Group 1 / Group 2).
* **Leave (Вихід):** Можливість скасувати реєстрацію на курс, що автоматично оновлює особистий графік.

**Для Адміністраторів (Admin)**
* **Керування розкладом:** Додавання (`add pair`) та видалення (`remove pair`) занять із глобального розкладу.
* **Валідація:** Система запобігає дублюванню занять та перевіряє наявність вільних аудиторій.

## Структура проєкту
```
.
├── users.json            # данні загальних тестів
├── schedule.json         # данні з фіксованою густиною
├── subjects.json         # данні з фіксованою кількість вершин
├── admin.json            # данні з фіксованою кількість вершин
│
├── class_Login.py        # система аутентифікації та реєстрації
├── class_User.py         # логіка студента: запис, відписка
├── class_Admin.py        # логіка адміністратора: добивити або прибрати пару
├── class_Visual.py       # малювання інтерфейсу та розкладу в консолі
├── class_WeekSchedule.py # Обробка даних розкладу
├── main.py               # Запускає логін та головне меню
│
├── helper_function.py    # допоміжні функції для зручної роботи
├── configuration.py      # Налаштування, константи та шляхи до баз даних
└── README.md             # Цей файл
```
